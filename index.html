Simulador de materiales para construcción

Este simulador calcula una estimación básica de materiales necesarios para construir una vivienda en Argenina 2025/2026, en función de los metros cuadrados, el tipo de ladrillo seleccionado (común u hueco) y el costo de las bolsas de cemento y arena ingresados por el usuario.

Toda la interacción se realiza mediante cuadros de diálogo (prompt, alert, confirm) y la consola de JavaScript del navegador. Para usarlo, abrí este archivo en el navegador, presioná F12 (o inspeccionar) y mirá la pestaña "Consola" antes de recargar la página.

Proyecto realizado como Entrega N° 1 del curso de JavaScript de Coderhouse.
